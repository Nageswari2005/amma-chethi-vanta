# 🍽️ Amma Chethi Vanta

## Traditional Andhra Restaurant Website

Amma Chethi Vanta is a professionally designed and fully responsive restaurant website developed to showcase the rich flavors and cultural heritage of authentic Andhra cuisine. The project provides a modern digital experience for customers while preserving the warmth and tradition of homemade Andhra food.

The website is designed with a clean and intuitive user interface, enabling visitors to explore menu offerings, restaurant ambience, customer reviews, and contact information with ease. It demonstrates strong front-end development skills and modern web design practices.
